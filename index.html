// Questions

Good to know.

Where does my monitoring data go?

Nowhere but your own infrastructure. MON1TOR runs entirely on your Docker host — metrics, alerts and logs live in your database, on your machine. The only thing that ever leaves is a one-off licence validation — never your data.

What do I need to run it?

A box that runs Docker — a NAS, a small VPS, a cloud instance. The whole stack comes up with a single Docker Compose command, and each machine you monitor gets a lightweight agent that registers itself.

Is there really no per-host pricing?

Correct. Base and Pro are one flat one-time price whether you watch three machines or three hundred. No tax for having more servers — that's exactly what makes the cloud platforms painful at scale.

What's the difference between Base and Pro?

Base is complete core monitoring — every machine's metrics, health score, anomaly detection, fleet heatmap and Telegram alerts. Pro adds the predictive and outward-facing layer: days-until-full forecasting, uptime & SSL checks, cron heartbeats, container insights, public status pages, 130+ alert channels (via Apprise) and weekly PDF reports. Buy Base and upgrade to Pro later for the difference (€50).

One-time, not subscription?

One-time. Pay once — €49 Base or €99 Pro — and it's yours, on your own infrastructure, for good. No monthly fees, no per-host charge, updates for your tier included. 14-day money-back if it's not for you.

Can I start on Base and move to Pro?

Yes. Buy Base now; whenever you want the Pro features, upgrade for the €50 difference — no need to buy Pro from scratch.
